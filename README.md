# Lower-level-design-probems

Collection of lower level design (LLD) Java practice projects.

## Repository Notes

- Source files are split so each top-level type is in its own `.java` file.
- All projects under `*/src` compile with plain `javac`.

Compile any project:

```bash
javac <ProjectFolder>/src/*.java
```

Run a project main:

```bash
java -cp <ProjectFolder>/src Main
```
